Project Overview
This repository contains a professional Sunburst Diagram visualization created for Schoen Champlin, a strategic consulting firm specializing in advanced data visualization and business intelligence solutions. The visualization analyzes organizational spending breakdown by department and expense type.
Visualization Details

Tool Used: RAWGraphs (https://rawgraphs.io/)
Chart Type: Sunburst Diagram
Visualization Method: Hierarchical nested structure
Aggregation: SUM of amounts by department and expense type
Image Format: PNG (512x512 pixels)

Data Structure
The visualization is based on organizational spending data across 6 departments and 20 expense categories:
Departments & Total Spending:

Marketing: $113,351 (Content, Events, Advertising, Social Media)
Operations: $100,077 (Supplies, Equipment, Maintenance, Utilities)
Sales: $129,550 (Travel, Client Entertainment, Commission)
IT: $122,400 (Software Licenses, Hardware, Cloud Services, Consulting)
HR: $55,700 (Recruitment, Training, Benefits)
Finance: $44,100 (Auditing, Software)

Data Points: 20 expense entries
Total Organizational Spending: $565,178
Business Context
